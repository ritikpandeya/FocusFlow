## **Description**
FocusFlow is a **professional-level, offline web application** that enhances productivity using the **Pomodoro technique**.  
It allows users to **manage tasks, track study time, and stay motivated**.  
Fully frontend, works **offline**, no backend or API needed.

---

## **Features**

- **Pomodoro Timer:** 25/5-minute sessions with animated circular countdown and session tracking  
- **Task Manager:** Add, complete, and delete tasks; tasks saved in LocalStorage  
- **Study Tracker:** Progress bar showing total focused time  
- **Motivational Quotes:** Random quotes every 30 seconds  
- **Dark/Light Mode:** Toggle between themes for comfortable usage  
- **Ambient Focus Sound:** Optional background sounds (e.g., rain, library)  
- **Mobile-Responsive & Animated UI:** Smooth interface with hover effects and circular timer
